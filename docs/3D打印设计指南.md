# 机器人3D打印结构设计 - 小白入门指南

## 一、什么是3D打印？

3D打印 = 把数字模型"打印"成实物

| 打印方式 | 材料 | 特点 | 适合 |
|----------|------|------|------|
| **FDM (最常见)** | PLA/ABS塑料 | 便宜、简单 | 初学者 |
| **光固化 (SLA)** | 树脂 | 精度高、表面光滑 | 精细件 |
| **SLS** | 尼龙粉 | 强度高 | 功能件 |

**推荐**：从 FDM 开始，设备 300-1000 元

## 四、设计流程 (以Fusion 360为例)

### 第一步：画草图

```
1. 打开Fusion 360 → 新建项目
2. 点击"创建草图" → 选择平面
3. 用"直线""圆""矩形"工具画2D图
4. 添加尺寸约束（标注长度）
```

### 第二步：拉伸/旋转

```
1. 选中草图 → 点击"拉伸"
2. 输入厚度（如10mm）
3. 变成3D实体
```

### 第三步：挖孔/切割

```
1. 画一个圆形草图
2. 点击"拉伸" → 选择"切除"
3. 输入孔径
```

### 第四步：组装

```
1. 点击"装配" → "新建组件"
2. 画多个零件
3. 用"约束"功能组装
```

## 五、机器人结构设计要点

### 5.1 主体设计

```
┌─────────────────────────────────┐
│         设计原则                │
├─────────────────────────────────┤
│ 1. 先大后小：先画整体，再画细节 │
│ 2. 壁厚均匀：防止打印变形      │
│ 3. 预留空间：留出电路板、电池   │
│ 4. 考虑散热：电机、芯片要散热   │
│ 5. 方便组装：考虑螺丝位置      │
└─────────────────────────────────┘
```

### 5.2 连接方式

| 连接方式 | 优点 | 缺点 |
|----------|------|------|
| 螺丝连接 | 牢固、可拆卸 | 需要打孔 |
| 卡扣连接 | 不用螺丝 | 容易断 |
| 胶水 | 快速 | 不可拆卸 |
| 热熔螺母 | 牢固 | 需要预埋 |

### 5.3 避坑指南

```
❌ 不要有悬空部分 (>45°需要支撑)
❌ 壁厚不要 <1.5mm (会裂)
❌ 孔径要留余量 (打印会缩小)
✅ 圆角设计 (防止应力集中)
✅ 打印方向选好 (影响强度)
```

## 六、设计示例：机器人头部

### 第一步：画外形

```
1. 新建草图 → 画圆 (直径80mm)
2. 拉伸 → 高度60mm
3. 圆角处理 → R5mm
```

### 第二步：挖显示屏孔

```
1. 画一个小圆 (50×30mm)
2. 放置在正面合适位置
3. 拉伸 → 切除
```

### 第三步：画眼睛

```
1. 画两个更小的圆
2. 切除 → 挖孔
```

### 第四步：加颈部接口

```
1. 画矩形凸起
2. 预留螺丝孔 (M3)
```

## 七、打印流程

### 步骤

```
1. 导出模型 → STL格式
2. 切片软件 → Cura/PrusaSlicer
3. 设置参数
   - 层高: 0.2mm (精度)
   - 温度: 200-210°C (PLA)
   - 速度: 50mm/s
4. 预览打印时间/耗材
5. 导出GCode → 拷贝到SD卡
6. 打印机操作
   - 调平平台
   - 装耗材
   - 开始打印
```

### 参数建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 层高 | 0.2mm | 精度与速度平衡 |
| 壁厚 | 1.2mm | 强度足够 |
| 填充 | 20% | 节省材料 |
| 温度 | 200°C | PLA标准 |
| 热床 | 60°C | 防翘边 |

## 八、后处理

### 打印完成后

```
1. 冷却后取下 (小心刮刀)
2. 去除支撑材料
3. 打磨表面 (砂纸从粗到细)
4. 组装测试
```

### 打磨顺序

```
粗砂纸(400) → 细砂纸(600) → 更细(800/1000) → 抛光
```

## 常见问题

**Q: 打印失败怎么办？**
A: 检查平台是否调平、层高是否合适、温度是否正确

**Q: 模型太大打印不下？**
A: 切分成多个零件分别打印，再组装

**Q: 哪里下载模型？**
A: Thingiverse、Printables、Sketchfab

**Q: 需要多少钱？**
A: 入门设备 500 元 + 耗材 100 元足够
