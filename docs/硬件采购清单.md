## 十、采购建议

### 推荐店铺

| 类别 | 推荐平台 |
|------|----------|
| RK3588 核心板 | 淘宝/闲鱼 "RK3588 核心板" |
| 摄像头/麦克风 | 淘宝 "USB 摄像头模组" |
| 电机/舵机 | 淘宝 "TT马达" / "MG996R" |
| 传感器 | 淘宝 "DHT22" / "VL53L0X" |
| 3D打印 | 淘宝/拼多多 "3D打印模型" |

### 注意事项

1. **RK3588 方案替代**：如预算有限，可选用 Jetson Nano (约 700元) 或 Orin Nano (约 1500元)
2. **摄像头**：可选带麦USB摄像头，可省去独立麦克风阵列
3. **3D外壳**：建议先买散件测试，后续根据需要定制
4. **电池**：购买带保护板的成品锂电池组
